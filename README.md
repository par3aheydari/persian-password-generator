# Persian Password Generator

A Persian password generator with a clean user interface, allowing you to select password length, character types (uppercase, lowercase, numbers, symbols), and displaying random security facts.

## Features

- Generate passwords with custom length
- Choose character types: uppercase, lowercase, numbers, symbols
- Display generated password
- Copy password to clipboard
- Show random security facts to highlight the importance of strong passwords
- Modern, Persian-language interface

## Usage

1. Download or clone the repository:
```bash
git clone https://github.com/username/persian-password-generator.git
```
2. Open index.html in your browser.

3. Select the desired password length and character options, then click "Generate Password".

4. Click "Copy" to copy the generated password to clipboard.
